Формат параметров консоли:

Требуемый формат:

file input output1 output2
или
random amount output1 output2

где
input - путь к файлу с входными данными
output1 - путь к файлу с выводом содержимого контейнера до сортировки
output2 - путь к файлу с выводом содержимого контейнера после сортировки
amount - целое число от 1 до 10000, количество случайно сгенерированных сущностей

Все файлы с данными должны существовать на момент запуска программы (даже файлы вывода)

Формат входного файла:

Первая строка: amount - целое число от 1 до 10000, количество записей в файле
Последующие amount * 3 строк: каждая группа из 3 строк образует одну запись следующего формата:
type - тип растения, целое число от 1 до 3, 1 - дерево, 2 - куст, 3 - цветок
name - имя растения, строка
parameter - параметр, уникальный для типа, целое число для деревьев, целое число от 1 до 12 для кустов, целое число от 1 до 3 для цветов

В случае несоответсвия одного из параметров формату или недостаточного числа строк программа сообщит об ошибке, выходные файлы будут созданы как если бы контейнер был пустым
